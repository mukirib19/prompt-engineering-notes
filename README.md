# prompt-engineering-notes

A working repository of prompt engineering notes, templates, and frameworks built from real experience writing and evaluating AI-generated content. This is not a theoretical overview of prompt engineering — it is a practical resource for writers, content strategists, and anyone who works with large language models and wants more consistent, usable outputs.

The prompts here have been written, tested, and refined through actual use. Each one includes a clear purpose, the full prompt text, and guidance on when to use it.

---

## What is inside this repository

- **prompt-library.md** — a collection of 8 fully written prompt templates covering the most common content writing and strategy tasks. Each prompt includes its purpose, the full text ready to use, and notes on when and how to apply it.

More templates and frameworks will be added over time as they are tested and refined.

---

## Who this is for

This repository is useful for:

- Content writers who want to get better results from AI writing tools
- Content strategists building repeatable AI-assisted workflows
- AI platform evaluators who need reference prompts for testing model outputs
- Anyone learning prompt engineering from a practical, writing-focused perspective
